Akordiyon (Accordion)

netio.css, Türkçe sınıf isimleri kullanan, hafif ve modern bir CSS framework'üdür. Sıfır bağımlılık ile çalışan ve kolay öğrenilen bir yapıya sahiptir.
dist/netio.css dosyasını projenize dahil edin, ardından Türkçe sınıf isimlerini HTML elemanlarınıza ekleyin. Örneğin: class="buton buton-birincil"
HTML etiketine data-tema="koyu" ekleyerek koyu temayı aktif edebilirsiniz. CSS değişkenleri üzerinden tüm renkleri özelleştirmek de mümkündür.

Sekmeler (Tabs)

netio.css, alternatif olarak tasarlanmış, tamamen Türkçe sınıf isimleri kullanan bir CSS framework'üdür.

12 kolonlu grid, responsive tasarım, 50+ bileşen, koyu tema desteği, utility sınıfları ve daha fazlası.

<link rel="stylesheet" href="dist/netio.css"> satırı ile projenize ekleyebilirsiniz.

Tema & Renk Yönetimi

CSS değişkenleri ile kolay tema özelleştirme. Light/Dark mod desteği.

Tema Değiştirme Yöntemleri
<!-- 1. HTML attribute ile --> <html data-tema="koyu"> <!-- 2. CSS sınıfı ile --> <body class="tema-koyu"> <!-- 3. JavaScript ile --> <script> document.documentElement.setAttribute('data-tema', 'koyu'); </script>
Özel Renk Tanımla
:root { /* Ana renkleri değiştirmek için */ --birincil: #8b5cf6; /* Mor ton */ --birincil-koyu: #7c3aed; --birincil-acik: #ede9fe; /* Yazı tipini değiştirmek için */ --yazi-tipi: 'Poppins', sans-serif; /* Yuvarlaklığı değiştirmek için */ --yuvarlaklık: 0.75rem; }
