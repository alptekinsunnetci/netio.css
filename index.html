Akordiyon (Accordion)

netio.css, Turkce sinif isimleri kullanan, Bootstrap benzeri, hafif ve modern bir CSS framework'udur. Sifir bagimlilk ile calisan ve kolay ogrenilen bir yapiya sahiptir.
dist/netio.css dosyasini projenize dahil edin, ardindan Turkce sinif isimlerini HTML elemanlariniza ekleyin. Ornegin: class="buton buton-birincil"
HTML etiketine data-tema="koyu" ekleyerek koyu temayi aktif edebilirsiniz. CSS degiskenleri uzerinden tum renkleri ozellestirmek de mumkundur.

Sekmeler (Tabs)

netio.css, Bootstrap'e alternatif olarak tasarlanmis, tamamen Turkce sinif isimleri kullanan bir CSS framework'udur.

12 kolonlu grid, responsive tasarim, 50+ bilesen, koyu tema destegi, utility siniflari ve daha fazlasi.

<link rel="stylesheet" href="dist/netio.css"> satiri ile projenize ekleyebilirsiniz.
